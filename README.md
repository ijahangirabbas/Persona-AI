# Persona AI

An AI-powered chat application that simulates conversations with two popular Indian software educators:

- Hitesh Choudhary
- Piyush Garg

## Features

- Persona-based AI chat
- Switch personas
- Temporary conversation memory
- Responsive UI
- Backend API
- Vercel deployment

## Tech Stack

Frontend
- React
- Vite
- CSS

Backend
- Node.js
- Express

LLM
- OpenAI Compatible API

## Project Structure

frontend/
backend/
docs/

git clone ...

npm run dev

npm run dev
